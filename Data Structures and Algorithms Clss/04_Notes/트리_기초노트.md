## 이진 트리
### 순회

#### 깊이 우선 순회: 스택을 이용하여 구현
- 전위 순회(preorder traversal)
```python
preorder(T)
    if(T != NULL):
        visit T.data
        preorder(T.Llink)
        preorder(T.Rlink)

    end preorder()
```
- 중위 순회(inorder traversal)
```python
inorder(T)
    if(T != NULL):
        preorder(T.Llink)
        visit T.data
        preorder(T.Rlink)
        
    end preorder()
```
- 후위 순회(postorder traversal)
```python
postorder(T)
    if(T != NULL):
        preorder(T.Llink)
        preorder(T.Rlink)
        visit T.data

    end preorder()
```
