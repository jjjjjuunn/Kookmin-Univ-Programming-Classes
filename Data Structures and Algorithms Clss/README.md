## 💡 학습 팁

### 중간고사 대비
1. **연결리스트 마스터하기**
   - 단순/이중 연결리스트 차이점
   - Stack/Queue 구현 방식
   - 노드 삽입/삭제 연산 손으로 그려보기

2. **알고리즘 시간복잡도 이해**
   - 이진 검색: O(log n)
   - 정렬: 삽입 O(n²), 병합/퀵 O(n log n)

### 기말고사 대비
1. **트리 순회 완벽하게**
   - 전위/중위/후위 순회 손으로 그려보기
   - BST 삭제 3가지 경우 이해

2. **해시테이블 충돌 해결**
   - 개방형: DELETED 마커 역할
   - 폐쇄형: 체이닝, 부하율, 재해싱

3. **스택 응용**
   - 후위표기법 변환 알고리즘
   - 연산자 우선순위 처리

## 🚀 사용 방법

1. **주제별로 찾기**: 폴더 구조를 따라 원하는 주제의 폴더로 이동
2. **시험 범위 확인**: 각 파일 상단에 `📚 중간고사` 또는 `📖 기말고사` 표시 확인
3. **복습 노트 활용**: `05_Notes/` 폴더의 마크다운 파일로 빠른 복습
4. **실습 코드 실행**: `.py` 파일은 바로 실행, `.ipynb`는 주피터 노트북으로 실행
